Rules

Standard Rules

Set-up

1. Separate Playing Cards from Set-up Cards.

2. Set all 4 Color Cards in a separate place.

3. Assign a Color Card to each player.

4. Shuffle Number Cards and randomly pick one and place in on top of the Color Cards, this will be the goal value each color has to be equal in order to win.

5. Shuffle Playing Cards and give each player 3 cards, set the deck face down.

6. Randomly assign a first player.

How to Play

Starting from the first each player will pick 1 Playing Card from the top of the Playing Card Deck or from The Pile.

In a turn a player must either place a card from his hand or remove a card from play.

A card can be placed before or after any other card.

When a card is removed from play it must be placed on a separate deck called The Pile, the player who removed a card must also discard one from his hand and place it on top of The Pile.

After placing/removing a card, the set of instructions in play are read by the players from left to right (Players must define which side is which at the beggining of the game, this cannot be changed midgame).

Each card has an instructions an all follow the same rules, with the exception of Loop Cards which can move cards to the right of the loop to a separate vertical set of instructions.

If a Loop Card is placed inside a loop it must be indented to the right

A player wins if after reading the set of instructions in play his color is equal to the goal value, when this happens the game ends.

Turns are allocated clockwise

Variants

Free For All

The game follows almost the same rules as the Standard Rules, the only difference is that a goal value is assigned to each color instead of all colors.

Set In Stone

The game follows almost the same rules as the Standard rules, the only difference is that once a card is played it can never be removed from play, so The Pile doesn't exist in this variant.

Loop Card

A Loop Card allows a player to repeat a set of instructions, the number of repetitions is defined by the number in the Loop Card. When the card is set the player chooses 0 to N consecutive cards to the right that will be part of the loop, those cards are placed below the Loop Card in their respective order.

Plus Card

A Plus Card adds 1 to the value of the color in the background of that card.

Minus Card

A Minus Card subtracts 1 to the value of the color in the background of that card.

Equal Card

An Equal Card sets the value of the color in the background of that card to the value of the color in the middle circle of that card.

All Plus Card

An All Plus Card adds 1 to the value of all 4 colors (Blue, Green, Red, Yellow).

Color Card

Color Cards are used as a reference of the value each color has to be equal in order to win the game. In the Standard Rules all colors have the same goal value, in some variants this is not the case.

Number Card

Number Cards are used to define the value each color has to be equal in order to win the game. In the Standard Rules a random Number Card is picked at the beggining of a game and is placed on top of all 4 Color Cards. In variants where players do not share goal value a Number Card is randomly picked and placed on top of each color.
